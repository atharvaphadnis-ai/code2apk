# 🌌 Axiom — AI Document Analyzer & Mind Map Generator

![Axiom](public/axiom.png)

**Axiom** is an open-source, AI-powered web app that lets you:

- 📂 Upload documents (`.txt`, `.pdf`, `.png`)
- 🔍 Get structured AI analysis — Title, Summary, Key Points, Insights
- 🧩 Generate beautiful mind maps from any prompt using **Pollinations AI**
- ⚙️ Bring your own API key (OpenRouter or Groq) — nothing is hardcoded

> Made with ❤️ by **Atharva Phadnis**

---

## ✨ Features

- ChatGPT-style dark UI
- Drag & drop file upload
- Streaming AI responses
- OCR for images (Tesseract.js)
- PDF text extraction (pdf-parse)
- Mind map preview via Pollinations
- Copy-to-clipboard for outputs
- Fully responsive design
- **"Set Save Folder"** — choose where uploaded files are stored
- **"Download App Files"** — grab the full source from the app itself

## ⚙️ Configuring an AI Provider

Open the app → click the **⚙️ Settings** icon (top-right) and choose a provider:

### 🟢 Groq (recommended — fastest & free tier)

1. Sign up at [https://console.groq.com](https://console.groq.com)
2. Create an API key
3. Paste it in **Settings → API Key**
4. Model is locked to `qwen/qwen3.6-27b` for Groq users (best quality/speed)

### 🟣 OpenRouter (any model you want)

1. Sign up at [https://openrouter.ai](https://openrouter.ai)
2. Create an API key
3. In Settings, choose **OpenRouter** and enter:
   - **API Key**
   - **Model name** (e.g. `anthropic/claude-3.5-sonnet`, `openai/gpt-4o-mini`, `meta-llama/llama-3.1-70b-instruct`)
   - **Base URL** (defaults to `https://openrouter.ai/api/v1`)

### 🔵 Using another provider (OpenAI, Together, DeepInfra, Fireworks, Anyscale…)

Any **OpenAI-compatible** `/chat/completions` endpoint works out of the box.

1. Choose **OpenRouter** in the provider dropdown (it's the generic OpenAI-compatible mode).
2. Set the **Base URL** to your provider's endpoint, for example:
   - OpenAI → `https://api.openai.com/v1`
   - Together → `https://api.together.xyz/v1`
   - DeepInfra → `https://api.deepinfra.com/v1/openai`
   - Fireworks → `https://api.fireworks.ai/inference/v1`
   - Anyscale → `https://api.endpoints.anyscale.com/v1`
   - Local Ollama → `http://localhost:11434/v1`
3. Enter your **API Key** and **Model Name**.

If your provider uses a different auth scheme, tweak the `Authorization` header in `public/app.js`.

## 🧠 How It Works

| File type | Pipeline |
|-----------|----------|
| `.txt`  | Read directly |
| `.pdf`  | `pdf-parse` extracts text |
| `.png`  | `tesseract.js` OCRs text |

Extracted text is sent to your configured LLM with a structured prompt asking for:

- **Title**
- **Summary**
- **Key Points**
- **Insights / Explanation**

Responses are **streamed** back token-by-token, just like ChatGPT.

Mind maps are generated by sending the prompt to Pollinations AI's image endpoint.
